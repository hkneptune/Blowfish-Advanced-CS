Encrypt

By pressing this button Blowfish Advanced CS starts to encrypt all the files and folders you have selected.

If you select a folder every file in it and files in subfolders (in which match the file mask and the attributes) are going to be encrypted.

Please remember that encrypted documents cannot be loaded with their associated applications anymore. For that the program offers two methods for renaming the files plus the possibility to add a special extension to the filenames

Do not encrypt anything that your system needs to run correctly, e.g. initialization or system files!

Before starting the encryption process a key must be provided, which can either be a password, a key disk or combinations of both, entered in the key input dialog.
