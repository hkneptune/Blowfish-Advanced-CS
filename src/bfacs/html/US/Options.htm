Options

The options dialog allows you to do the fine tuning of the program to make it work for you as best as possible.

Split in four main categories it offers you a good chance for finding the necessary options as fast as possible:

Security
Configuration
File Handling
Miscellaneous

Changes in the options it will only be valid for the current runtime. If you want to fix them permanently click on the Apply button. It will be enabled as soon as a storable change was made.
