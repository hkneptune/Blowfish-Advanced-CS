Browser

The browser of Blowfish Advanced CS allows you to navigate with ease through the file system on your computer and on networks you are connected to.

Please check out the following features:

  • Use your own favorites for easier and faster browsing.
  • A double click will open a file (view or work with encrypted files)
  • Dragging and dropping of files and folders causes a move process.
  • Dragging and dropping of files with the ALT key held down causes a copy process.
  • Activate the popup menu with a right mouse button click.
  • Sort the columns by clicking on the headers (possible only in Details Style).
