Button Bar and Browser Tools

The button bar allows you to start file actions intuitively and quickly, to call the options dialog, to exit the program, to get the information about the program and to call the help file.

The browser tools assist you to navigate quickly and very comfortably through your file system, to create new folders or set the browser viewing style.

All actions of the button bar and the browser tools are also reflected in the main menu. Check the key shortcuts (e.g. Ctrl+E to encrypt) to improve your working speed with Blowfish Advanced CS.
