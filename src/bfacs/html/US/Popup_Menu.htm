Popup Menu

The popup menu can be activated by clicking with the right mouse button into the browser.
It provides functionality for file handling and a quick access to the most common operations of Blowfish Advanced CS.

Depending on the item under your mouse pointer some menu entries might not be enabled.
Please remember that all menu items are equipped with key shortcuts, too.

Encrypt...

Equals the menu entry or the button in the bar.

Decrypt...

Equals the menu entry or the button in the bar.

Wipe

Equals the menu entry or the button in the bar.

Work With...

Equals the menu entry or the button in the bar.

Copy...

Copies all selected files to the path you have declared in the destination dialog.

Move...

Moves all selected files to the path you have to declare in the destination dialog.

Rename...

Opens the dialog for renaming files and folders.

Delete

Removes all selected files and folders. Please remember that items deleted with Blowfish Advanced CS will not go to the Recycle Bin

Select...All

If you want to select all files and folders in the current path.

Select...Files

If you want to select all files in the current path.

Select...Folders

If you want to select all folders in the current path.

Select...Encrypted Files

If you want to select all files with the .BFA extension or recognized as encrypted by the scanner.

Select...Unencrypted files

If you want to select all files without the .BFA extension or recognized as not encrypted by the scanner.

Select...Files Containing String...

If you want to select files containing a defined string within their names.

Format...

Allows you to format a drive, depending on its type. The dialog is identical to the one in Windows Explorer. On older Windows versions this command might not be available and permanently be disabled.
