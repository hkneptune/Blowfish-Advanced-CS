Progress Dialog

In the progress window you can watch how the work on your files is going.

Depending on the action you will be shown:

  • the name of the current file(s)
  • the size of the current file
  • the number of the current processed file and a total
  • the number of the current wipe loop
  • a progress bar showing you the overall progress
  • a progress bar showing you the progress on the current file

Use the Pause/Continue button to temporarily stop the process, e.g. if you need your system's power at 100% for other applications.

You can also Minimize the running application to get a free desktop.

The Cancel button will abort the process after a confirmation.
