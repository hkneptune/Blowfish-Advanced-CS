View

If a single selected file in the browser carries the .BFA extension or was recognized by the scanner as encrypted Blowfish Advanced CS ask you for a key to decrypt the file before it will be started with its associated application.

If the key was the right one the file will be decrypted in the temporary path for viewing and its name an path added to an internal list. Before Blowfish Advanced CS terminates the temporary files will be wiped using the current selected method.

Remember to use View only for readonly purposes. If you want to change a file's content use Work With instead.
