Wipe

By pressing this button Blowfish Advanced CS starts to wipe all the files and folders you have selected.

If you select a folder every file in it and files in subfolders (in which match the file mask an the attributes) are going to be wiped.

You can choose between different wipe methods for faster execution or more overwrite cycles.

Please be careful - wiping means destroying!

If a file has been wiped (except of being deleted only) its original data will be lost forever and cannot be restored even with low-level disk editing utilities!
