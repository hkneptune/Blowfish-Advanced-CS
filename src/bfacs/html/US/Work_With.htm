Work With

Blowfish Advanced CS allows you to to work with encrypted files.

Working means the following three steps:

  1. decrypting the selected file
  2. launching the associated application and waiting for the user to finish the work
  3. reencrypting the file with the decryption key and renaming it to the old name.

This feature automates most of the annoying work which results from handling encrypted files. The key must be entered only once, no other actions are necessary afterwards, too. The preservation of the old filename is also useful, especially for backup purposes of random renamed files.
