Job Report

The job report window shows you the results of one or multiple executed jobs in a table view. You can check out immediately where and what errors or problems did occur. Multiple Jobs are shown in separated sheets, showing their titles makes it easy to select the right one. Additionally you see the time a job needed for execution in the status line of every sheet.

Save...

You can save the whole job report in a text file by clicking this button. The report will be stored properly formatted, thus is easy to read afterwards.

Create Job File...

By clicking this button you can store the executed job in a jobfile, so you can repeat it easily later.

First you will be asked for the name and location of the jobfile to create. If you select an already existing jobfile you can either overwrite it or append the job. As the last step you must enter a title for the job, making it recognizable when executed posteriori.
