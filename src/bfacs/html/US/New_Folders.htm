New Folders

Click this button to create new folders in the current path.

In opposition to the standard folder creation in the Windows Explorer, you may not only enter a simple name for the new path in the appearing dialog box, but also provide nested paths. This is very useful if you want e.g. to create a whole path chain, like "docs\personal\august" with one single action.

Additionally you can create multiple paths in one strike. Each line in the input box represents a name for a new path. Empty lines are ignored. To close the dialog quickly press ALT+O (or Tab and Enter).
