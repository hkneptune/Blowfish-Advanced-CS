Miscellaneous

Temporary Path for Viewing

For viewing encrypted files Blowfish Advanced CS decrypts them to a temporary directory, and wipes them when you exit the program. Here you can set a different temporary path. E.g. if you just want to have a look at your encrypted files at an other person's computer you may set the temporary path to the floppy drive to let the files be written temporarily to your own disk material.

Key Disk File

Here you can define the path and name of the keydisk used by Blowfish Advanced CS. Only files placed on removable disks should be used, anything different doesn't make sense at all, of course.

Make Key Disk

Here you can open the dialog to make a key disk.

Clear Key Hashes for the Auto Confirmation

You can delete all key hash entries created by the auto confirmation. here.

Maximum Number of Errors

If the number of errors during a job reaches a definite limit Blowfish Advanced CS will ask if you want to continue. You can set this limit here.

Registry Setup

Here you can decide which level of integration in the Windows environment you prefer. You might want to associate file types (.BFA for encrypted files and .BFJ for jobfiles) with the program. There is also the possibility to activate context menu extensions, which means adding a new entry in the popup menu of the Windows Explorer. By clicking on the Update button you change the current configuration, modifying the registry of Windows. This is also useful if you want to remove Blowfish Advanced CS from a system: just uncheck the two switches and update. All registry entries associated with the program will be cleared.
