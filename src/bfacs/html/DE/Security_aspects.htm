Sicherheitsaspekte

Wenn Sie ein Passwort wählen, so müssen Sie sich stets einen wichtigen Aspekt vor Augen halten: jedes Passwort kann durch eine sogenannte Brute-Force-Attacke herausgefunden werden, was nichts anderes bedeutet, daß alle möglichen Passwörter durchprobiert werden.

Wenn Ihr Passwort aus drei Buchstaben besteht und man davon ausgeht, daß nur Zeichen von "a" bis "z" verwendet wurden, dann gibt es maximal 26 * 26 * 26 = 17.576 Möglichkeiten die zur Erkennung des Passwortes probiert werden müssen. Mit einem optimierten Suchprogramm, auf einem schnellen Computer ausgeführt, würde so Ihr Passwort innerhalb weniger Millisekunden gefunden werden.

Wenn Sie jedoch nur 2 Zeichen mehr verwenden, so muß man ca. 11 Millionen Kombinationen testen, was um einiges mehr Zeit benötigt, selbst auf einem schnellen Rechner. Entspricht das Passwort (selbst wenn nur im Ansatz) einem gängigen Wort, so kann es durch eine Wörterbuchattacke aber wiederum sehr viel schneller gefunden werden.

Die Regel ist einfach: je mehr Zeichen Sie verwenden, um so mehr Kombinationen bestehen, um so schwieriger ist eine verschlüsselte Datei zu brechen. Mit einem Schlüssel von 6 Bytes (entspricht 48 Bits), welcher aus einem genügend langen Passwort erzeugt wurde, existieren 256 ^ 6 = 281.474.976.710.700.000.000 (281,5 Trillionen) Kombinationen. Nimmt man an, daß ein schneller Computer ungefähr eine Million Schlüssel pro Sekunde testen kann, so wird es ca. 9 Jahre dauern um alle Kombinationen zu probieren. Noch ein Byte mehr und eine Brute-Force-Attacke wird extrem aufwendig.

Rechenleistung wird natürlich billiger mit der Zeit, Mikroprozessoren immer leistungsfähiger, so daß jede Sicherheitsrechnung stets von Jahr zu Jahr angepaßt werden muss.

Wie sollten Sie Ihr Passwort wählen?

Benutzen Sie keine gebräuchlichen Worte, so wie z.B. den Namen Ihres Ehepartners, Tochter, Sohn, Hund, Katze, Lover, Ihre Versicherungs-, Haus- oder Telefonnummer, die Zahlen Ihres Geburtstages oder -jahrs, nicht einmal in umgekehrter Reihenfolge. Sie können sicher sein, daß diese

Passwörter als erstes ausprobiert werden.

Wenn Sie keine Passwörter mit extra Zeichen wie z.B. "&%$*" verwenden wollen, so benutzen Sie bitte Passwörter, die mindestens 10 Zeichen lang sind. Benutzen Sie zudem Zahlen oder binäre Werte. Ein gutes Passwort z.B. ist ein Satz ohne Sinn, der aber leicht gemerkt werden kann, z.B. "Der Hund ist zu grün?".

Bitte erinnern Sie sich immer an die goldene Regel:

Vergessen Sie niemals Ihr Passwort!

Prägen Sie es sich gut ein!

Wenn Sie sich Passwörter nicht merken können, verwenden Sie Schlüsseldisks.

Es gibt keine Möglichkeiten Dateien wiederherzustellen, die durch Blowfish Advanced CS mit einem vergessenen Schlüssel von ausreichender Länge gesichert wurden. Wenn das Passwort oder die Keydisk verloren gehen sind Sie nicht mehr in der Lage Ihre Dateien zu entschlüsseln. Das Programm speichert die Schlüssel nirgends ab, weder in der verschlüsselten Datei noch sonst irgendwo. Die Schlüssel selbst werden sogar nach ihrem Gebrauch aus dem Speicher gelöscht.
