Fortschrittsanzeige

Im Fenster der Fortschrittsanzeige können den Verlauf der Arbeit an Ihren Dateien beobachten. Abhängig von der aktuellen Aktion wird folgendes angezeigt:

  • der Name der aktuellen Datei(en)
  • die Größer der aktuellen (Eingabe)Datei
  • die Nummer der aktuell bearbeiteten Datei und die Gesamtanzahl
  • die Nummer des aktuellen Überschreibzyklus
  • ein Fortschrittsbalken für den gesamten Prozess
  • ein Fortschrittsbalken für die Bearbeitung der aktuellen Datei

Verwenden Sie die Schaltfäche Pause/Fortfahren um den laufenden Prozess anzuhalten, z.B. um 100% Systemressourcen für andere Applikationen zur Verfügung zu haben.

Sie können außerdem die laufende Applikation Minimieren um den Desktop wieder frei zu haben.

Die Schaltfäche Stop bricht den Prozess nach einer Bestätigung ab.
