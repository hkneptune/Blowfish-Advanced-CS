Konfiguration

Die Konfiguration enthält Schalter für die verschiedenste Zwecke für Blowfish Advanced CS. Anstelle unkontrollierter Verteilung in irgendwelchen Dialogen sind sie hier in eine sortierten Liste schnell und bequem zu finden.

Abgeschnittene Dateien akzeptieren

Die Software weigert sich standardmäßig unvollständige Dateien (verursacht z.B. durch einen abgebrochenen Download) zu entschlüsseln. Aktivieren Sie diesen Schalter um die Überprüfung zu umgehen und soviel Daten als möglich rekonstruieren zu lassen.

Attribute bleiben erhalten

Wenn dieser Schalter aktiv ist werden die Attribute von verschlüsselten Dateien nicht zurückgesetzt, die Originalattribute bleiben erhalten. So bleiben versteckte Dateien z.B. weiterhin versteckt. Bitte beachten Sie, daß die originalen Attribute trotzdem in der verschlüsselten Datei gespeichert werden.

Auswahl im Stile eines Webbrowser

Wenn Sie es bevorzugen, daß der Auswahl so wie ein Webbrowser aussieht und sich dementsprechend verhält, so sollten Sie diese Option aktivieren. Alle Einträge sind dann unterstrichen und einzelne Mausklicks werden als Doppelklicks behandelt.

Automatische Auswahlaktualisierung

Die Software ist in der Lage zu erkennen, wann Dateien oder Ordner im aktuellen Pfad gelöscht, erzeugt oder verändert wurden. Tritt ein solcher Fall ein, dann wird die Auswahl automatisch auf den neuesten Stand gebracht.

Automatisches Anordnen

Die Einträge werden in der Auswahl automatisch angeordnet wenn sich die Fenstergröße ändern sollte.

Beenden nachdem ein Shelljob ausgeführt wurde

Wenn Sie einen Job (aus einer Jobdatei oder durch direkt eingegebene Objekte) über den Windows Explorer oder durch Drag & Drop von Objekten über das Programmsymbol gestartet haben, so können Sie das Programm dazu veranlassen nach der Ausführung zu beenden. Die einzige Ausnahme ist das Betrachten einer verschlüsselten Datei (die sofort wieder verschwinden und die Applikation zum Betrachten zurücklassen würde), wobei hier das Programm weiter läuft.

Bereits verschlüsselte Dateien übergehen

Ist dieser Schalter aktiviert, so prüft Blowfish Advanced CS jede Datei, ob Sie schon verschlüsselt wurde. Ist dies der Fall, so wird kein zweites Mal verschlüsselt. Bei deaktiviertem Schalter entfällt diese Prüfung. Dann wird jede Datei verschlüsselt, egal in welchem Zustand sie sich befindet. Bitte beachten Sie, daß wenn Sie Dateien mehrmals verschlüsseln Sie diese auch mehrfach in umgekehrter Reihenfolge entschlüsseln müssen.

Betrachtete Dateien dürfen existierende überschreiben

Betrachtete verschlüsselte Dateien werden in ein temporäres Verzeichnis entschlüsselt. Obwohl solche Verzeichnisse normalerweise keine Dateien von Wert enthalten können Sie durch Deaktivieren dieses Schalters ein Überschreiben existierender Dateien vermeiden. Bitte beachten Sie, daß bei mehrfacher Betrachtung derselben Datei eine Überschreibsituation permanent auftritt. Wenn Sie sich nicht sicher sind, welche Methode nun die bessere ist können Sie zum Betrachten Ihr eigenes temporäres Verzeichnis verwenden (s. Verschiedenes).

Cache-Speicher für Lesen und Schreiben umgehen

Wenn Daten während der Ver- und Entschlüsselung gelesen und geschrieben werden, so werden sie wie gewöhnlich durch das Windows Betriebssystem gecacht. Obwohl dies für die tägliche Arbeit von Vorteil ist, so kann es gerade beim Bearbeiten einer großen Menge von Daten allgemein eher bremsend wirken. Sie können hiermit das Programm dazu bringen, durch den Cache-Speicher direkt hindurch auf die Platte zu schreiben, was die Ausführungsgeschwindigkeit erhöht. In manchen Fallen kann dies es aber auch zu negativen Effekten führe, speziell beim Bearbeiten vieler kleiner Dateien.

CRC32 Fehler ignorieren

Um Datenintegrität zu gewährleisten berechnet Blowfish Advanced CS eine CRC32-Prüfsümme und vergleicht sie mit derjenigen, welche in der verschlüsselten Datei gespeichert wurde. Sind diese Prüfsummen ungleich, so wird die originale Datei nicht wiederhergestellt, um den Benutzer eine Möglichkeit zu geben das Problem zu beheben, z.B. einen Fehler auf dem Datenträger zu korrigieren oder eine Datei neu zu übertragen. Durch diesen Schalter können Sie die Integritätsprüfung umgehen. Danach wird eine Datei immer entschlüsselt, egal ob die Daten beschädigt sind oder nicht. Bitte benutzen Sie diese Einstellung mit Vorsicht! Fertigen Sie von der verschlüsselten Datei eine Sicherheitskopie an, um die Entschlüsselung eventuell wiederholen zu können.

Doppelklick entspricht "Arbeiten mit"

Wenn dieser Schalter deaktiviert ist führen Doppelklicks (inklusive bei Drag & Drop einer einzelnen, verschlüsselten Dateien auf das Programmfenster) zur Betrachtung der verschlüsselten Datei. Wenn Sie statt dessen bearbeiten möchten, so aktivieren Sie diese Option. Dies führt auch zur einer Änderung beim der Reaktion auf Doppelklicks auf .BFA Dateien im Windows Explorer: anstelle eines Auswahldialogs wird sofort damit begonnen, mit der Datei zu arbeiten.

Erweiterung .BFA anhängen

Sie können verschlüsselte Dateien mit Blowfish Advanced CS verknüpfen. Dazu müssen die Dateien die Endung .BFA besitzen. Durch diesen Schalter wird dann die Erweiterung an den Namen jeder verschlüsselten Datei angehängt, unabhängig davon, ob sie umbenannt wurde oder nicht. Wenn z.B. der originale Dateiname "MeinTagebuch.txt" war, so wird sie in "MeinTagebuch.txt.bfa" umbenannt.

Existierende Dateien überschreiben

Setzen sie diesen Schalter, wenn Sie wollen, daß die Software existierende Dateien ohne zusätzliche Bestätigung überschreibt.

Farbige Fortschrittsanzeige

Aktivieren Sie diesen Schalter, wenn Sie die Boxen im Fortschrittsdialog farbig haben wollen. Verschlüsselung führt zu einer roten, Entschlüsselung zu einer grünen, Vernichtung zu einer gelben, Umverschlüsselung zu einer blauen und das Löschen von freiem Dateispeicher zu einer seegrünen Einfärbung.

Flache Auswahlansicht

Ein Gimmick, um die Auswahl im neuen Stil von Windows 98 anzuzeigen. Kann unter Windows 95 und bei NT4 Systemen unter Umständen nicht immer funktionieren.

Gitterlinien

In der Detailansicht der Auswahl werden Gitterlinien verwendet.

Hinweise anzeigen

Hier legen Sie fest ob die Schaltflächen mit kleinen Tips versehen werden sollen. Wenn Sie mit Blowfish Advanced CS vertraut genug sind können Sie diese abschalten.

Hot Tracking

Der Mauszeiger wählt nach einer kurzen Verzögerung den Eintrag unter sich automatisch aus.

In der Taskleiste laufen lassen

Um einen schnellen Zugriff auf das Programm zu haben kann Blowfish Advanced in der Taskleiste verbleiben. Das Programm ist nach dem Minimieren immer noch komplett aktiv, Sie sind aber in der Lage es über das Menü des Taskleistensymbols zu beenden. Wenn Sie dieses Features nicht nutzen wollen, so können Sie es hier abschalten.

Jobreport anzeigen

Blowfish Advanced CS merkt sich jede ausgeführte Operation und zeigt die Ergebnisse danach im Jobreport an. Wenn Sie eine solche Zusammenfassung nicht sehen wollen, so deaktivieren Sie diese Option. Bitte beachten Sie daß, falls irgendwelche Probleme aufgetreten sind, der Jobreport trotzdem angezeigt wird.

Kompression erzwingen

Die Software wiederholt den kompletten Verschlüsselungsvorgang falls eine Datei sich als nicht komprimierbar herausstellt. Dadurch wird der Überhang des Kompressors nicht gespeichert. Obwohl dies Platz spart erhöht es doch den Zeitaufwand aufgrund der Wiederholung. Durch die Benutzung dieser Option werden Daten immer komprimiert, selbst wenn die letztendliche Datei dann grösser als die ursprüngliche ist (anhängig vom Kompressionsalgorithmus kann dies bis zu 12.5% mehr sein), und dadurch der Gesamtvorgang beschleunigt.

Konfigurationsdateikonflikte anzeigen

Falls die Anwendung die Konfiguration nicht erfolgreich in BFACS.INI speicher kann wird eine Fehlermeldung in einer Dialogbox angezeigt. Sie koennen dies abschalten um BFACS.EXE von einem Nur-Lesen-Gerät wie z.B. einer CD-ROM auszuführen. Bitte beachten Sie dass eine entsprechende Konfigurationdatei mit abgeschalteter Option jedoch immer noch benötigt wird.

Laufwerke nicht anzeigen

Falls Sie keine Laufwerke in der Auswahl angezeigt haben wollen, so können Sie diese Option aktivieren. Sie sind jedoch immer noch in der Lage den Pfad bzw. das Laufwerk zu wechseln.

Laufwerke voranstellen

Laufwerke erscheinen in der Auswahl ganz oben.

Leere Verzeichnisse entfernen

Falls komplette Ordner vernichtet wurden und danach leer sind, entweder durch einen Vernichtungsvorgang oder z. B. durch die Verwendung eines Zielpfades, so können Sie diese durch die Aktivierung dieser Option automatisch entfernen lassen. Bitte beachten Sie, falls Ordner noch Dateien enthalten sollten (aus welchen Gründen auch immer) diese dann nicht gelöscht werden.

Nur .BFA Dateien scannen

Der Scanner untersucht standardmäßig alle Dateien in der aktuellen Auswahl. In den meisten Fällen macht es jedoch nur Sinn Dateien mit der Erweiterung .BFA zu scannen. Dieses Verhalten kann durch die Aktivierung dieser Option eingestellt werden.

Operationen müssen bestätigt werden

Ist dieser Schalter aktiviert, so wird Sie Blowfish Advanced CS vor jeder Ausführung fragen, ob Sie diese auch wirklich starten möchten. Diese Option wurde implementiert, um erfahrenen Benutzern die Möglichkeit zu geben auf unnötiges Nachfragen zu verzichten. Dieser Schalter hat übrigens keinen Einfluß auf Vernichtungsprozesse, da diese zu gefährlich sind, um hier ein Risiko einzugehen.

Sicherheitsrelevante Informationen speichern

Blowfish Advanced CS legt seine Konfiguration in der Datei BFACS.INI ab, die sich im selben Verzeichnis wie BFACS.EXE befindet.

Falls Sie diesen Schalter deaktivieren werden folgende Informationen nicht in BFACS.INI gespeichert:

  • alle Pfade der Auswahl (werden gelöscht)
  • der letzte Arbeitspfad (anstelle dessen wird der temporäre Pfad von Windows eingetragen)
  • der Pfad der Schlüsseldiskette (auf Standard zurückgesetzt)
  • der letzte Zielpfad (wird gelöscht)
  • das Temporärverzeichnis zum Betrachten von Dateien (s. letzter Arbeitspfad)
  • alle verwendeten Algorithmen werden auf den Standard (Blowfish) zurückgesetzt
  • alle Dateien in der Arbeitsliste

Symbole (und Typen) von gescannten Dateien ersetzen

War der Scanner in der Lage von einer verschlüsselten Datei Informationen auszulesen, so werden der originale Name und die Größe in der Auswahl angezeigt (Detailansicht). Für eine bessere Übersicht können Sie sich zusätzlich die originalen Symbole und die Typen der verschlüsselten Dateien anzeigen lassen, indem Sie diese Option aktivieren. Bitte beachten Sie, daß die originalen Symbole nur bei Dateien, nicht bei ausführbaren oder speziellen Dateien, angezeigt werden.

Verschlüsselte Dateien schreibschützen

Mit dieser Option werden aller Dateien nach abgeschlossenem Verschlüsselungsvorgang schreibgeschützt. Dies verhindert z.B. daß wenn in anderen Applikationen versehentlich eine verschlüsselte Datei geöffnet wird diese nur betrachtet, nicht aber verändert werden kann (was evt. zu Ihrer Zerstörung führen könnte).

Versuchen 8.3 kompatibel umzubenennen

Wenn Dateien verschlüsselt werden besteht die Möglichkeit, daß der Name der verschlüsselten Datei nicht vom darunterliegenden Dateisystem unterstützt wird, z.B. bei der Arbeit auf Servern, die nur DOS-Dateinamen unterstützen. Durch die Aktivierung dieser Option wird versucht Dateinamen kompatibel zum 8.3 (DOS Format) zu erzeugen. Bitte beachten Sie, daß dies nicht für die Entschlüsselung funktioniert. Falls Sie keine Probleme mit Ihrer Umgebung haben, ignorieren Sie diese Option und lassen Sie sie abgeschaltet.

Zeitstempel bleiben erhalten

Ist dieser Schalter aktiviert, so ändert die Software Datum und Zeit von verschlüsselten Dateien nicht. Genau wie die Attribute werden die Zeitstempel jedoch auch in den verschlüsselten Dateien gespeichert.
