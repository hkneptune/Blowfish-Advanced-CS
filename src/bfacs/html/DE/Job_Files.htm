Jobdateien

Verschlüsseln und Entschlüsseln von Dateien ist mit Blowfish Advanced CS einfach, nichtsdestotrotz sind es immer wieder die gleichen Aktionen, wenn Dateien und Ordner jeden Tag verschlüsselt, entschlüsselt, vernichtet oder umverschlüsselt werden müssen.

Um diese täglichen Sicherheitsarbeiten zu automatisieren bietet Blowfish Advanced CS Ihnen Jobdateien an. Ein Job ist nicht mehr als eine Beschreibung was wann wie (und wohin) zu erledigen ist.

Wenn Sie z.B. eine Anzahl von Dateien verschlüsselt haben, so können Sie diese Aktion in einer Jobdatei abspeichern. Benutzen Sie dafür den Menüeintrag im Jobreport. Wenn Sie den Job erneut ausführen wollen, so klicken Sie einfach auf die Jobdatei. Dann werden Sie nicht mehr die Dateien auswählen müssen, Sie werden nicht mehr die ursprünglichen Optionen und den korrekten Verschlüsselungstreiber einstellen, noch werden Sie den Zielpfad festlegen müssen - Sie werden lediglich das Passwort eingeben bzw. die Schlüsseldiskette einlegen. Sie können sogar einen Job zu einer bereits existierenden Jobdatei hinzufügen. Wird diese dann ausgeführt, so wird jeder Job ihn ihr abgearbeitet.

Sie können z.B. eine Jobdatei erzeugen die alle Entschlüsselungsaufträge für Ihre vertraulichen Dateien für die tägliche Arbeit in Ihrem Büro enthält. Jeden Morgen müssen Sie nun nur die Jobdatei starten und das Passwort eingeben - das ist alles. Auf gleichem Wege können Sie das Gegenstück zu dieser Jobdatei erzeugen, welche Ihre Dateien verschlüsselt. Fügen Sie dann noch Jobs zum Aufräumen hinzu, z.B. um alle temporären Dateien aus dem Internet zu vernichten, und schon haben Sie die Jobdatei, die Sie beim Verlassen des Büros dann starten.

Wenn Sie Dateien verknüpft haben, so können Sie direkt auf die Jobdatei klicken ohne Blowfish Advanced CS vorher zu starten. Wie wäre es z.B. die Jobdatei für die tägliche Entschlüsselung auf dem Desktop oder im Ordner Autostart zu plazieren?
