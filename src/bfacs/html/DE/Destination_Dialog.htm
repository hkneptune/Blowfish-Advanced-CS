Zielpfad

Dieser Dialog bietet Ihnen nicht nur die Möglichkeit einen Zielpfad festzulegen, sondern auch aus einer Liste bereits eingegebener Pfade auszuwählen.

Benutzen Sie die Schaltfläche Löschen um einen nicht mehr benötigten Eintrag in der Liste zu entfernen.

Verwenden Sie die Schaltfläche Suchen... um einen neuen Pfad auszuwählen.

Existiert ein Pfad nicht, so kann Blowfish Advanced CS ihn für Sie erzeugen.
