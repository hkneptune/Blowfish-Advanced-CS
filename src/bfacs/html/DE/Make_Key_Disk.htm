Schlüsseldisk erzeugen

In diesem Dialog können Sie eine sichere Schlüsseldisk(ette) erstellen.

Als ersten Schritt schieben Sie bitte eine leere Disk(ette) in das Laufwerk, welches unter den Optionen angegeben wurde.

Dann wählen Sie eine geeignete Grösse für die Schlüsseldatei. Bitte beachten Sie, daß die Schlüsselgrösse hier in Bits gemessen wird, und somit nur in Schritten zu 8 erhöht oder verringert wird (solange man noch keine einzelnen Bits in Dateien speichern kann). Der höchste Wert ist übrigens 1024, was für Schlüssel zur symmetrischen Verschlüsselung bereits mehr als zuviel ist.

Um einen wirklich brauchbaren Schlüssel zu erzeugen ist es notwendig, daß der Zufallszahlengenerator (Yarrow wird für diese Zwecke empfohlen) mit genügend zufälligen Daten zuvor gefüttert wird, bevor er die Schlüsselbytes erzeugt. Durch Bewegen der Maus über dem Fenster oder Tippen auf der Tastatur können Sie solche Zufallsdaten erzeugen.

Bitte beachten Sie, daß das Fenster für solche Zwecke in der Größe veränderbar ist.

Nachdem genügend Zufallsdaten gesammelt wurden erscheint ein Dateidialog, in welchem Sie den Namen und den Pfad der Schlüsseldatei ändern können. Die Erzeugung der Datei selbst dauert (normalerweise) weniger als eine Sekunde.

Bitte entfernen Sie die Schlüsseldiskette sofort. Nachdem Sie sich entschieden haben, sie zu verwenden, sollten zumindest eine Sicherheitskopie von ihr bzw. der Schlüsseldatei anlegen.

Bitte achten Sie darauf, daß Sie bereits existierende Schlüsseldateien nicht versehentlich überschreiben!
