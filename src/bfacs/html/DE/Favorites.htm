Favoriten

Auf die gleiche Weise, wie Sie interessante Links im Internet als Bookmarks speichern, können Sie eine Liste Ihrer am häufigsten verwendeten Pfade in Blowfish Advanced CS anlegen.

Wenn Sie den aktuellen Pfad als Favoriten abspeichern wollen, klicken Sie auf den Menüeintrag Favoriten...Aktuellen Pfad hinzufügen. Der Pfad wird dann permanent im der Liste im Menü Favoriten verfügbar sein. Das Wechseln zu komplett unterschiedlichen Pfaden ist nun nur noch eine Sache von einer Sekunde anstelle von umständlichem Navigieren.

Sie können Ihre Favoriten bearbeiten, indem Sie auf den Menüeintrag Favoriten...Verwalten... klicken. In der erscheinenden Dialogbox können Sie dann nicht mehr erwünschte Einträge löschen. Mehrfachauswahl ist möglich, falls Sie mehrere Favoriten auf einmal löschen wollen.
