Freien Dateispeicher löschen

Wenn Daten in einer Datei abgelegt werden, so reserviert das Betriebssystem so viele Cluster (der kleinsten logischen Einheit auf einem Laufwerk) auf dem Medium wie nötig um alle Dateidaten aufzunehmen. Ein Cluster hat eine feste Größe, z.B. 512 Bytes bis hin zu 32 kB, abhängig vom Dateisystem. Die meisten Dateien passen von der Größe her natürlich meistens nie genau in dieses Raster.

Den nicht verwendeten Speicherplatz nennt man Slack und kann unter Umständen vertrauliche Daten bereits gelöschter Dateien enthalten.

Sie können mit diesem Menüeintrag den Slack der ausgewählten Dateien (und Dateien in Unterordnern) löschen.
