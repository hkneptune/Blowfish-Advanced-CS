Schlüsselspeicher

Wenn mit verschlüsselten Dokumenten gearbeitet wird, so wechseln die meisten Benutzer nur sehr selten ihre Schlüssel. Somit wird es nötig, denselben Schlüssel zum Ver-, Ent-, Umverschlüsseln und Betrachten jedesmal zeitaufwendig neu einzugeben.

Blowfish Advanced CS bietet hierfür eine Lösung: haben Sie einen Schlüssel einmal eingegeben, so können Sie ihn für eine bestimmte Zeit (oder auch solange Sie wollen) in einem internen Speicher ablegen. Damit der Schlüssel nicht versehentlich in die Auslagerungsdatei gerät wird er in einem speziellen Speichertyp angelegt.

Mit einem so gespeicherten Schlüssel müssen Sie nur noch bestätigen, anstelle ihn neu einzugeben. Für manche Aktionen, z.B. für das Betrachten verschlüsselter Dateien ist nicht einmal mehr eine Bestätigung nötig.

Einen Schlüssel im Hintergrund stets parat zu haben sollte allerdings mit ein wenig Vorsicht genossen werden. Nur Sie sollten während dieser Zeit Zugriff auf Ihren Computer haben. Sollten Sie dies nicht mehr haben, empfiehlt es sich den Schlüsselspeicher manuell zu löschen.
