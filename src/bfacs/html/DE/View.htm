Betrachten

Trägt eine einzeln ausgewählte Datei die Erweiterung .BFA oder wurde Sie vom Scanner als verschlüsselt identifiziert, so fragt Blowfish Advanced CS nach einem Schlüssel, bevor die originale Datei mit Ihrer verknüpften Applikation gestartet wird.

War der Schlüssel korrekt, so wird die Datei in das temporäre Verzeichnis zum Betrachten entschlüsselt und Name sowie Pfad in eine interne Liste aufgenommen. Bevor Blowfish Advanced CS beendet werden diese temporären Dateien dann vernichtet (mit der augenblicklich ausgewählten Methode).

Bitte beachten Sie, daß Sie Betrachten nur für Lesezwecke verwenden. Wenn Sie den Inhalt der Datei ändern wollen, so verwenden Sie Arbeiten mit anstelle.
