Jobreport

Das Jobreport Fenster zeigt Ihnen die Ergebnisse von einem oder mehrerer Jobs in Tabellenform an. Sie können sofort erkennen wo und welche Fehler aufgetreten sind. Mehrfachjobs werden auf getrennten Seiten dargestellt, dazugehörige Titel machen es einfach, den gesuchten Job zu finden. Zusätzlich sehen Sie die von einem Job benötigte Zeit in der Statuszeile auf jeder Seite.

Speichern...

Sie können den gesamten Jobreport in einer Textdatei speichern, indem Sie auf diese Schaltfläche klicken. Der Report wird für eine bessere Lesbarkeit sauber formatiert.

Job speichern...

Mit dieser Schaltfäche können Sie einen ausgeführten Job in einer Jobdatei speichern, und somit später leicht wiederholen.

Zuerst werden Sie nach dem Namen und dem Pfad der Jobdatei gefragt. Wählen Sie eine bereits existierende Jobdatei aus, so können Sie diese entweder überschreiben oder den Job anhängen. Als letzter Schritt wird ein Titel für den Job verlangt, um ihn später besser wiedererkennen zu können.
