Freien Speicherplatz löschen

Mit diesem Dialog können Sie den gesamten freien Speicherplatz auf einem oder auch mehreren Laufwerken löschen. Das Programm erzeugt hierzu eine temporäre Datei, bis das Laufwerk komplett voll ist. Somit werden alle ungenutzten Dateicluster überschrieben.

Bitte beachten Sie, dass das Löschen von freien Speicherplatz jedliche Versuche zur Wiederherstellung von Dateiinhalten, inklusive regulärer Rettungsmassnahmen, unmöglich macht!

In den Einstellungen können Sie festlegen was und wie Blowfish Advanced CS zu löschen hat.

Nicht verzögertes Schreiben

Das Erzeugen von mehreren GB großen Dateien macht jeden Caching-Mechanismus von Windows überflüssig. Im Gegenteil kann Caching den Prozess verlangsamen oder Ihren Computer vollständig blockieren, bis alle Buffer auf die Disk geschrieben wurden. Um solche Probleme zu vermeiden können Sie diesen Schalter aktivieren. Dies sollte in den allermeisten Fällen die bessere Lösung sein.

Keine Meldungen

Nachdem der freie Speicher eines Laufwerks gelöscht wurde erscheint eine Meldung. Die temporäre Datei wird erst nach dem Schließen dieses Dialogs gelöscht. Obwohl Blowfish Advanced CS alle Buffer vorher speichert hat kann es somit in einigen Fällen helfen, Windows (oder selbst der Hardware von Festplatten) etwas zusätzliche Zeit zu geben um den Rest der Daten zu schreiben. Wenn Sie jedoch den freien Speicher von mehreren Laufwerke löschen wollen können solche Meldungen eher lästig werden und eine unbeaufsichtigte Ausführung schwierig machen (z.B. während einer Kaffeepause). Durch die Aktivierung des Schalter wird dies verhindert.

Niedere Priorität

Sie können Blowfish Advanced CS dazu bewegen während dem Löschprozess seine Priorität zu verringern, was es Ihnen ermöglicht mit anderen Applikationen (die normalerweise von einer höheren Priorität sind) ohne merkbare Verzögerungen zu arbeiten. Bitte beachten Sie, daß am Ende des Löschprozesses auf dem jeweiligen Laufwerk kein Speicherplatz mehr frei ist, was besonders beim Speichern in anderen Applikationen zu Fehlern führen kann.

Wo löschen?

Sie können der Software mitteilen, wo zu löschen ist: entweder auf dem aktuellen Pfad, auf allen Laufwerken (CD-ROMs ausgenommen, Wechseldatenträger jedoch eingeschlossen) oder auf allen Festplatten.

Löschen mit

Eine temporäre Datei kann entweder Nullen oder Zufallsdaten enthalten. Bei Zufallsdaten können Sie zwischen der Erzeugung eines einzigen Blocks und dessen wiederholter Verwendung (statische Zufallsdaten) oder der Ausgabe von kontinuierlichen Zufallsdaten entscheiden, welches die langsamste Methode ist, da das Programm die Daten andauernd erzeugen muss, nicht nur während des Startvorgangs. Die Verwendung von Zufallzahlen ist bei komprimierten Laufwerken auf jeden Fall zu empfehlen.

Klicken Sie auf die Schaltfläche Start um mit dem Löschen zu beginnen. Wenn die Größe des freien Speichers (auf dem aktuellen Laufwerk) erkannt werden konnte, so erscheint auf der rechten Seite ein Fortschrittsbalken. Andernfalls wird die Anzahl der geschriebenen Bytes in der Statusbar angezeigt. Der Prozess kann jederzeit durch die Schaltfläche Abbrechen beendet werden.
