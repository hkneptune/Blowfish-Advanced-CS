Neue Ordner

Klicken Sie auf diese Schaltfäche um neue Ordner im aktuellen Pfad anzulegen.

Im Gegensatz zur gewohnten Vorgehensweise im Windows Explorer können Sie in der Dialogbox nicht nur einen neuen Pfad, sondern eine ganze Pfadreihe angeben. Dies ist sehr nützlich, wenn Sie mehrere Pfade auf einmal anlegen wollen, z.B. "Dokumente\Persönlich\August".

Zusätzlich können Sie mehrere Pfade auf einmal anlegen. Jede Zeile in der Eingabe-Box repräsentiert einen neuen Pfad. Leerzeilen werden ignoriert. Um den Dialog schnell abzuschließen drücken Sie entweder ALT+O (oder Tab und Enter).
