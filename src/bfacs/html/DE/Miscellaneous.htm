Verschiedenes

Temporärer Pfad zum Betrachten

Um verschlüsselte Dateien betrachten zu können entschlüsselt Blowfish Advanced CS sie in ein temporäres Verzeichnis und vernichtet diese dann bei der Beendigung des Programms. Hier können Sie nun das temporäre Verzeichnis ändern. Wenn Sie z.B. nur einen Blick auf die verschlüsselten Dateien auf einem fremden Rechner werfen wollen, so können Sie den Pfad auf das Diskettenlaufwerk umlenken, um die temporären Kopien auf Ihrer eigenen Diskette zu haben.

Schlüsseldatei

Hier können Sie den Pfad und den Namen der Schlüssel(disk)datei festlegen, die von Blowfish Advanced CS verwendet werden soll. Logischerweise sollten nur Dateien auf auswechselbaren Datenträgern verwendet werden.

Schlüsseldisk erzeugen

Hier öffnen Sie den Dialog, um eine Schlüsseldisk(ette) zu erzeugen.

Prüfsummen für die automatische Bestätigung löschen

Sie können hier alle Prüfsummen löschen, die durch die Automatische Bestätigung erzeugt wurden.

Maximale Anzahl von Fehler

Wenn die Anzahl von Fehlern beim Ausführen eines Jobs ein bestimmtes Limit erreicht hat fragt Blowfish Advanced CS nach, ob Sie trotzdem fortfahren wollen. Diese Grenze kann hier festgelegt werden.

Systemeinstellungen

Hier können Sie entscheiden, inwieweit die Software mit der Windows Umgebung zusammenspielt. Sie können .BFA und .BFJ Dateitypen verwenden, d.h. mit Blowfish Advanced CS verknüpfen. Es besteht ebenso die Möglichkeit, Erweiterungen für das Kontextmenü des Windows Explorers zu aktivieren. Durch Klick auf die Schaltfäche Setzen werden die Änderungen in der Registrierdatenbank von Windows vorgenommen. Dies ist auch dann nützlich, wenn Sie Blowfish Advanced CS aus Ihrem System deinstallieren wollen: deaktivieren Sie hierzu beide Optionen und klicken Sie auf Setzen. Alle Einträge in der Registrierdatenbank, die mit der Software zu tun haben, werden dann gelöscht.
