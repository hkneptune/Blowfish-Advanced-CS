Dateien

Umbenennungsmethode

Hier können Sie auswählen, welche Methode zur Umbenennung von verschlüsselten Dateien verwendet werden soll, um die originalen Dateinamen somit zu verbergen. Die Umbenennungsmethoden haben keinen Einfluß auf das Anhängen der .BFA Erweiterung an eine verschlüsselte Datei.

Umbenennung mit Benutzermaske

Sie können einen Kopf und eine Erweiterung für umbenannte Dateien angeben. Blowfish Advanced CS wird dann eine fortlaufende Dezimalzahl zwischen diesen zwei Teilen einfügen, um Kollisionen von Dateinamen zu vermeiden.

Umbenennung mit zufälligen Zeichen

Dateien werden mit 8 zufällig ausgewählten Zeichen umbenannt. Danach sehen die Dateinamen äußerst zufällig aus.

 

Kompression

Dateien der folgenden Typen nie komprimieren

Einige Dateitypen verwenden bereits ein komprimiertes Format, was jeden Versuch sie weiter zu verkleinern überflüssig macht. Sie können Blowfish Advanced CS vorgeben, welche Dateitypen bei der Komprimierung zu ignorieren sind.

Hierfür geben Sie die Erweiterungen ein, getrennt durch Kommas. Mit einem Satz der am häufigsten vorkommenden, komprimierten Dateien können Sie immer mit eingeschalteter Kompression arbeiten, ohne unnötige Wiederholungen zu riskieren. Die Software ist bereits mit einer Auswahl von bekannten Dateitypen vorbereitet, die meisten hiervon Archiv- und Multimediaformate.
